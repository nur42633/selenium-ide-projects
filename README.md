# Selenium IDE Projects

This repository contains a collection of automated test cases and practical tasks developed using **Selenium IDE**. The projects demonstrate various UI test scenarios, workflows, and automation techniques.

## Contents

- Sample test cases for web application UI testing
- Test scripts covering common Selenium IDE commands and features
- Workflow automation examples
- Notes and best practices for Selenium IDE usage

## How to Use

1. Import the `.side` files into Selenium IDE.
2. Run or customize the tests as per your requirements.
3. Extend the test suite with additional cases or commands.

## Tools Used

- Selenium IDE
- Web Browsers (Chrome, Firefox)
